Place your images in incarnate-dice with integer file names to use this helper. 

Hitting a number will advance the image by one and put the key in a list below
Hitting backspace will remove the last list item and go back one image, allowing you to correct a mistake.

After you are done copy-paste the list into a text file for analysis.
Note this is the only way to save your data! Make sure you don't hit refresh below you do this!
